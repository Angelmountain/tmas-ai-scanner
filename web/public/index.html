Query Syntax (TMV1-Query)

Use Trend Micro Vision One query syntax:

hostName:(*.example.com) - Match hostnames
app:(RDP OR SSH) - Match applications
dstPort:(22 OR 3389) - Match destination ports
ruleName:(*SSH*) - Match detection rules
suid:(administrator) - Match user accounts
(productCode:pdi OR productCode:xns) - Filter by sensor type
